### 关键提示/结论
- TanStack Router 嵌套路由机制：
  - 父路由文件（如 `projects.tsx`）必须使用 `<Outlet />` 渲染子路由
  - 子路由文件放在同名文件夹内（如 `projects/index.tsx`、`projects/$projectId.tsx`）
  - 路由配置由 `routeTree.gen.ts` 自动生成
- API 响应格式：后端直接返回对象时，前端不应使用 `res.data`
